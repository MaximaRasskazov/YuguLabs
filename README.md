# 🚀 Yugu Server - Микросервис Авторизации

## 🛠 Технологии
* **Язык:** Go (Golang)
* **Фреймворк:** Gin HTTP web framework
* **База данных:** SQLite + GORM
* **Безопасность:** JWT (Access & Refresh токены), bcrypt (хеширование паролей)
* **Валидация:** go-playground/validator (с кастомными правилами)

## ⚙️ Добавьте .env

   ```env
   ACCESS_TOKEN_TTL=60
   REFRESH_TOKEN_TTL=10080
   MAX_ACTIVE_TOKENS=5
   JWT_SECRET=super_secret_yugu_key_123
   ```

## 📡 API Эндпоинты

### Публичные маршруты
| Метод  | Маршрут | Описание |
| :--- | :--- | :--- |
| `POST` | `/api/auth/register` | Регистрация нового пользователя |
| `POST` | `/api/auth/login` | Авторизация и получение пары токенов |
| `POST` | `/api/auth/refresh` | Обмен старого Refresh-токена на новую пару ключей |

### Защищенные маршруты
| Метод  | Маршрут | Описание |
| :--- | :--- | :--- |
| `GET`  | `/api/auth/me` | Получить профиль текущего пользователя |
| `GET`  | `/api/auth/tokens` | Получить список всех активных сессий/устройств |
| `POST` | `/api/auth/out` | Выйти с текущего устройства (требует `refresh_token` в теле) |
| `POST` | `/api/auth/out_all` | Завершить все сеансы на всех устройствах |

## 🧪 Как удобно тестировать?

Для тестирования мы подготовили файл `api.http`. 
Если вы используете **VS Code**, установите расширение **REST Client**. Если **GoLand** — поддержка встроена.

1. Откройте файл `api.http`.
2. Нажмите кнопку `Send Request` над запросом **Login**.
3. Токены автоматически сохранятся во внутренние переменные редактора.
4. Теперь вы можете отправлять любые другие запросы (например, `/me` или `/tokens` в один клик), и редактор сам подставит нужный токен!
